# Daily Bible Verse App

A Flutter app that displays a Verse of the Day and allows users to generate random Bible verses.

## Features
- 1,100+ verses
- Verse of the Day
- Random verse generator
- Bookmark your favorite verses
- Offline Bible JSON
- Daily reminders
- Light / Dark / System theme
- Clean, minimal UI

## Tech Stack
- Flutter + Dart
- Firebase Authentication
- Cloud Firestore
- Local JSON assets

## Screens
- Home (Daily Verse)
- Saved Verses
- Settings
