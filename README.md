# Daily Bible Verse App

A Flutter app that displays a Verse of the Day and allows users to generate random Bible verses.

## Features
- 1,100+ verses
- Verse of the Day
- Random verse generator
- Offline Bible JSON
- Clean, minimal UI

## Tech Stack
- Flutter
- Dart
- Local JSON assets

## Screens
- Home (Daily Verse)
- Saved Verses
- Settings

## Getting Started
```bash
flutter pub get
flutter run
